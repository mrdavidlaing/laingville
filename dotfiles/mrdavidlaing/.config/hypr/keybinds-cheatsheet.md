# Hyprland Keybinds Cheatsheet

## Window Management
- `Super + C` - Close active window
- `Super + V` - Toggle floating window
- `Super + F` - Toggle fullscreen
- `Super + P` - Toggle pseudo-tiling
- `Super + J` - Toggle split direction

## Navigation
- `Super + Arrow Keys` - Move focus between windows
- `Super + Ctrl + Left/Right` - Switch workspace left/right

## Workspaces
- `Super + 1-9,0` - Switch to workspace 1-10
- `Super + Shift + 1-9,0` - Move window to workspace 1-10
- `Super + Mouse Scroll` - Cycle through workspaces

## Window Movement
- `Super + Mouse + Left Click` - Move window
- `Super + Mouse + Right Click` - Resize window

## Launchers
- `Super + R` - Application launcher (rofi)
- `Super + Q` - Terminal (gnome-console)
- `Super + E` - File manager (thunar)
- `Super + W` - Web browser (vivaldi)
- `Super + G` - Image editor (gimp)

## System
- `Super + M` - Exit Hyprland
- `Super + Shift + R` - Reload Hyprland configuration
- `Super + H` - Toggle cheatsheet (Super + Shift + H to create first time)

## Media Controls
- `XF86AudioRaiseVolume` - Volume up
- `XF86AudioLowerVolume` - Volume down
- `XF86AudioMute` - Mute toggle
- `XF86MonBrightnessUp` - Brightness up
- `XF86MonBrightnessDown` - Brightness down
- `XF86AudioPlay` - Play/pause media
- `XF86AudioNext` - Next track
- `XF86AudioPrev` - Previous track

## Special Workspaces
- `Super + S` - Toggle special workspace (magic)
- `Super + Shift + S` - Move window to special workspace

## Additional Apps
- `Super + T` - Roblox emulator (Sober)
- `Super + K` - Minecraft launcher
